# Next.js Project

This project is a Next.js application that is currently live at [https://next-js-project-rno4.vercel.app](https://next-js-project-rno4.vercel.app).

## Technologies Used
- **TypeScript**: 94.6% of the codebase is written in TypeScript, providing static typing for improved development and maintenance.
- **JavaScript**: 4% of the code is in JavaScript, used for compatibility and flexibility.
- **CSS**: 1.4% of the project utilizes CSS for styling the components.

Next.js allows for server-side rendering and static site generation, enhancing performance and SEO for this web application.
